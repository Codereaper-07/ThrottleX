ThrottleX

Production-ready distributed rate limiter built with Node.js, Redis, Docker, Prometheus and Grafana.

Demonstrates distributed rate limiting with atomic Redis transactions and real-time monitoring — enforced consistently across every instance behind the load balancer, not just per-process.

Engineering highlights

Token Bucket Algorithm

Smooth, time-based refill that allows short bursts without abrupt window resets.

Redis Optimistic Locking

WATCH/MULTI/EXEC on isolated connections keeps concurrent bucket updates atomic.

Dockerized Deployment

One command starts the app, Redis, Nginx, Prometheus and Grafana together.

Prometheus Monitoring

Custom counters and histograms expose traffic, block rate and latency.

Grafana Dashboard

Auto-provisioned datasource and dashboard — no manual setup required.

Production-ready Backend

Graceful shutdown, centralized error handling, and fail-fast configuration.
